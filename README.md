## About The Project

This is a simple gin middleware to validate JWT tokens with a given
public key.

It is mostly inspired by [gin-jwt](https://github.com/appleboy/gin-jwt/blob/master/auth_jwt.go).

### Built With

- Golang
- [gin-gonic](https://github.com/gin-gonic/gin)
